Musethereal

Overview

This repository contains code for the MakeFashion project known as Musethereal. Musethereal is a dress modelled after the neuron. This repository contains the code for the conceptual dendrites, responding to the position of the accelerometer tilt.

Technology Stack

Musethereal uses:

  1. The Emotiv EPOC+ EEG Headset to pull activity from the brain

  2. An Android phone (we used the LG Nexus 5) connected to the EPOC+ via bluetooth and using the Emotiv community sdk to pull data from the headset

  3. felHR85's UsbSerial library was used to communicate data from the EPOC+ to the Arduino

  4. The Arduino Trinket Pro 3V uses Adafruit Neopixel RGB LEDs to represent the chasing, cascading sequences, corresponding to the accelerometer tilt on the arms and on the skirt of the dress as appliqu√©s.

  5. Adafruit Neopixel library was used to set RGB LEDs' sequences

  6. ADXL335 - Triple-Axis Accelerometer Breakout was used to read the analog value of the x, y and z direction of the hand gestures and output it to the LED strip as a chase sequence
